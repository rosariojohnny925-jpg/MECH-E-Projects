# Basic Components

## Overview

This repository contains a collection of CAD components that I have designed and modeled throughout various mechanical engineering projects. These parts represent foundational design work used in larger assemblies, prototypes, and engineering concepts developed during coursework and personal projects.

The models are provided in STEP format to ensure compatibility across multiple CAD platforms and to support future modifications, manufacturing preparation, and assembly integration.

## Purpose

The purpose of this repository is to maintain a library of reusable mechanical components and design elements that can be incorporated into future projects. By organizing these files in a central location, I can efficiently reference previous work, iterate on existing designs, and streamline the development process.

## Skills Demonstrated

* CAD Modeling
* Mechanical Design
* Design Iteration
* Engineering Documentation
* Component Development
* Assembly Preparation
* Manufacturing Considerations

## Software

These models were created using CAD software commonly used in engineering design and can be imported into most major CAD platforms through the STEP file format.
